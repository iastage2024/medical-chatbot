# ☁️ Déploiement

Ce projet est conçu pour être déployé sur **Google Cloud Run**.

---

## 🐳 Étapes prévues

1. Création d’un Dockerfile
2. Build de l’image
3. Push vers Artifact Registry
4. Déploiement Cloud Run
5. Secrets via variables d’environnement

---

## 🔄 CI/CD

Avec GitHub Actions :
- tests automatiques
- build Docker
- déploiement automatique

## ✅ Avantages Cloud Run

- Stateless
- Scalabilité automatique
- Paiement à l’usage
- Sécurisé
